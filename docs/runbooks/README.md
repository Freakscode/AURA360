# Runbooks AURA360

Esta carpeta consolida toda la documentación operativa que antes vivía en la raíz del repositorio. Usa los subdirectorios según el tipo de procedimiento:

| Carpeta | Descripción | Contenido principal |
|---------|-------------|---------------------|
| `deployment/` | Planes y checklists de despliegue para API, wellness intake y verificación en producción | `DEPLOYMENT.md`, `DEPLOYMENT_CHECKLIST.md`, `PRODUCTION_ENV_SETUP.md`, `DEPLOY_WELLNESS_INTAKE_QUICKSTART.md`, `VERIFICACION_DESPLIEGUES_RESUMEN.md`, etc. |
| `kafka/` | Documentación de la plataforma de mensajería (Kafka/Confluent) | `KAFKA_*`, `CONFLUENT_CLOUD_SETUP.md`, `MCP_*`, `QUICKSTART_KAFKA.md` |
| `setup/` | Guías de configuración de credenciales, Supabase e issues de permisos | `CREDENTIALS_COLLECTION_GUIDE.md`, `SUPABASE_SETUP.md`, `UV_PERMISSIONS_FIX.md` |
| `agents/` | Documentos específicos de asistentes internos | `CLAUDE.md` |

> Nota: si mueves nuevos runbooks fuera de la raíz, actualiza este índice para mantener una sola fuente de verdad.
